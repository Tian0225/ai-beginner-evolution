## 欢迎来到 AI 小白的进化之路！

这是一份专为**初学者**设计的 AI 工具实战教程。

### 为什么写这个教程？

我在使用 Claude Code 的过程中发现，官方文档对新手不够友好，很多概念（MCP、Skills、API）讲得太技术化。

我想用**通俗易懂**的方式，结合我的实战经验（B站带货、AI博主项目），让每个人都能学会使用这些强大的 AI 工具。

### 你将学到什么？

- ✅ **Claude Code 基础**：安装、配置、日常使用
- ✅ **MCP 服务器**：让 AI 连接外部工具（搜索、读文件、看图片）
- ✅ **Skills 系统**：自定义 AI 的能力，让它更懂你
- ✅ **API 集成**：调用 OpenAI、Kimi、Gemini 等服务
- ✅ **实战项目**：B站视频生成、自动化工作流等

### 适合谁看？

- 🎯 **有一定编程基础，但对 AI 工具不熟的朋友**
- 🎯 **想提升工作效率的程序员**
- 🎯 **对 AI 创业感兴趣的创业者**

### 学习路径

```
第1章：快速上手 → 第2章：MCP 服务器 → 第3章：Skills 系统 → 第4章：API 集成 → 实战项目
```
