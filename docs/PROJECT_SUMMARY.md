## 📊 项目概览

这是一个完整的 Claude Code 和 AI 工具链教程项目，从零开始教初学者如何使用 AI 工具提升工作效率。

### 核心数据

- **文档总数**：9 个页面
- **核心章节**：6 章
- **实战案例**：6 个真实项目
- **总字数**：约 50,000 字
- **代码示例**：100+ 个
- **图片数量**：20+ 张

### 技术栈

- **网站框架**：VitePress 1.6.4
- **版本控制**：Git
- **部署平台**：GitHub Pages
- **开发工具**：Claude Code
